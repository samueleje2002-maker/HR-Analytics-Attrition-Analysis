# 📊 HR Analytics Dashboard – Attrition Analysis

## 📌 Project Overview
This project focuses on analyzing employee attrition trends using Power BI. The dashboard provides insights into workforce demographics, salary distribution, departmental attrition, business travel impact, and other HR-related metrics.

## 🎯 Objectives
- Analyze employee attrition patterns
- Identify factors contributing to employee turnover
- Visualize HR metrics interactively
- Generate actionable business insights

## 🛠 Tools & Technologies Used
- Power BI
- Excel
- Data Visualization
- HR Analytics
- Business Intelligence

## 📈 Dashboard Features
- Employee Count
- Attrition Count
- Average Age
- Average Salary
- Average Tenure
- Attrition by Department
- Attrition by Education
- Attrition by Salary Slab
- Attrition by Marital Status
- Attrition by Business Travel

## 🔍 Key Insights
- Lower salary employees showed higher attrition
- R&D department had highest attrition
- Single employees had higher attrition rate
- Business travel affected employee turnover
